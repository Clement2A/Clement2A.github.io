French City Name Generator

Try it out: Click Here

This is quite an old code I had done while still doing my fullstack web developper master. I used already existing city names, and divided them by phonemes. I ended up with many syllables that when randomly put together, alongside some prefixes and suffixes, create some fairly believable french-sounding city names. I also added a few extra checks, to avoid cases like double vowels, or remove a consonent when 2 syllables starting and ending with one fuses.
