This is the first big Unreal project I got to work on at Objectif3D, alongside other students after having learned about the engine for about 2 months. After the game designers had made a prototype using Blueprints, we've worked for 3 months on it to make it as complete as possible

I've worked on quite a few elements in this project:

  • The tutorial gamemode, alongside all prompts reacting the the player's action
  • All world widgets, like the enemy health bars, the animated popup when an enemy is destroyed and the notifications when you first build something
  • The navgrid system, calculating a path for the enemies with A*, alongside a tool to easily generate the grid on the level and modify it
  • The building sytem,
  • Some UI elements, like the player's health bar, the building selection widget and the crosshair
