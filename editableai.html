Unity In-Game Editable AI

GitHub repo link: Click here

A small project of my own where I experimented with an AI that could be edited in-game by the player directly.

I've started this with a very basic finite state machine, to which I slowly added more and more features until the AI's behaviour could properly be changed by a player. To make it easier to use, I've also made an UI allowing for states to be created and deleted easily, alongside draggable to make everything easier to organise.

It is not done yet, I still need to finish the UI to edit the AI, and then get some placeholder states to test it afterward.
